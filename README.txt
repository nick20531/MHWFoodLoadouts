================================================================
  MHW Food Loadouts
  Version 1.0 | Monster Hunter World (Iceborne) v421810
================================================================

  Instantly apply food skill loadouts without visiting the
  canteen. No Cheat Engine required.

----------------------------------------------------------------
  REQUIREMENTS
----------------------------------------------------------------

  - Monster Hunter World: Iceborne (Steam, version 421810)
  - Windows 10 or 11
  - Must be loaded into the hub or a quest before attaching
    (not the main menu or character select screen)

----------------------------------------------------------------
  FIRST TIME SETUP
----------------------------------------------------------------

  Windows will warn the app is from an unknown publisher and
  may quarantine it. This is normal for any tool that writes
  to game memory — Cheat Engine has the same issue.

  Step 1 — Extract and run build file

  Step 2 — Unblock the .exe:
    1. Right-click MHW Food Loadouts.exe
    2. Click Properties
    3. At the bottom tick "Unblock" then click OK

  Step 3 — If Windows Defender quarantines it:
    1. Open Windows Security
    2. Go to Virus & threat protection > Protection history
    3. Find the blocked file and click Allow
    Or: add the folder to exclusions before running

  Step 4 — If attach fails with a permissions error:
    Right-click the .exe and select "Run as administrator"

----------------------------------------------------------------
  HOW TO USE
----------------------------------------------------------------

  1. Launch Monster Hunter World and load into the hub
     or a quest — NOT the main menu
  2. Run MHW Food Loadouts.exe
  3. Click "Attach to Game"
     Status should show: Attached ✓  Food struct found
  4. Select a loadout from the dropdown
  5. Click "Apply Loadout"

  Values take effect immediately. No need to eat at the
  canteen first.

  AUTO-APPLY
  ----------
  Toggle "Auto-Apply: OFF" to turn auto-apply on.
  The selected loadout reapplies every 3 seconds,
  keeping buffs active across zone loads and quest
  transitions automatically.

  Turn auto-apply off before visiting the canteen if
  you want to eat normally.

----------------------------------------------------------------
  MANAGING LOADOUTS
----------------------------------------------------------------

  + New    Create a new loadout via the built-in editor
  Edit     Modify the currently selected loadout
  Delete   Remove the currently selected loadout

  Loadouts are saved to config.json next to the .exe.
  Back this file up or share it to transfer your setup.

----------------------------------------------------------------
  TROUBLESHOOTING
----------------------------------------------------------------

  "Pointer step X failed"
    You need to be fully loaded into the hub or a quest.
    Wait for the loading screen to finish then try again.

  "Failed to open process" / attach does nothing
    Right-click the .exe and run as Administrator.

  Apply does nothing / wrong values
    Make sure game version is v421810 (fully patched
    Iceborne on Steam). Other versions are unsupported.

  Values reset after a zone load
    Turn on Auto-Apply — it reapplies every 3 seconds
    and catches any resets automatically.

  Windows Defender blocks the app
    See First Time Setup above.

  Game updated and app stopped working
    A game update may have changed the memory offsets.
    Check for an updated version of this app.

----------------------------------------------------------------
  NOTES
----------------------------------------------------------------

  - Only modifies food skill buffs in memory. Does not
    affect saves, items, equipment or anything else.

  - Values reset when the game is closed. Nothing is
    written to disk or save files.

  - Compatible with online play. Other players cannot
    see your food values. The tool does not interact
    with game servers in any way.

  - Requires game version 421810 (Iceborne fully patched).
